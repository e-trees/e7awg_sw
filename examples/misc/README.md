# AWG のステータスを読み取る

[read_awg_status.py](./read_awg_status.py) は，全ての AWG のステータスを読み取って出力するスクリプトです．
このスクリプトは，開発者向けのデバッグ用スクリプトです．

## 実行方法

以下のコマンドを実行します．

```
python read_awg_status.py [--ipaddr=AWG コントローラの IP アドレス]
```
